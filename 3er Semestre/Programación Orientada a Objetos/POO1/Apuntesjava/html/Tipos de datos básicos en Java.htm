Variables (definición)

Una variable es un lugar en memoria donde pueden guardarse valores; tienen un nombre un tipo y un valor.

 

Declaración de variables

Las declaraciones de variables consisten en un tipo y un nombre de variable:

 

Tipo nonVar;

 

o

 

Tipo nomVar1, nomVar2,..., nomVarN;

 

Ejemplos de declaraciones de variables

 

int edad;

string nombre;

boolean bandera;

int x, y, z;

 

Inicialización de variables en la declaración

 

Tipo nomVar=Valor;

o

Tipo nomVar1, nomVar2,...,nomVarN=Valor;

 

 

Ejemplos de inicialización de variables

 

int edad, tamano, numero=28;

String nombre=”Juan”;

boolean bandera=true;

char c=’a’;

Arreglos

Definición

Un arreglo sirve para almacenar una colección de elementos del mismo tipo. Para seleccionar un elemento del arreglo se utiliza un índice (en otras palabras se usa un indice como selector).

 

Declaración de arreglos

Los arreglos se pueden declarar de las dos formas siguientes:

 

Tipo nomArr[];

o

Tipo [] nomArr;

 

Ejemplos de declaración de arreglos

 

String words[];

Point hits[];

 

Declaración y asignación de memoria

La manera de declarar y asignar memoria a un arreglo es la siguiente

 

Tipo nomArr[] = new Tipo[tamano];

o

Tipo [] nomArr = new Tipo[tamano];

Ejemplos de la declaración y asignación de memoria

 

String [] nombres=new String[];

int [] temps=new int[99];

 

Ejemplo de inicialización de arreglos

 

String [] dias={“Lunes”,”Martes”,”Miercoles”,”Jueves”,”Viernes”,”Sabado”,”Domingo”};

Int n[]={32, 27, 64, 18, 95, 14, 90, 70, 60, 37};

 

Todos los arreglos tienen una variable de instancia llamada length la cual contiene el tamaño del arreglo.

 

Manejo de argumentos en la línea de comandos

Java guarda los argumentos en la línea de comandos en un arreglo de cadenas, el cual se pasa al método main

 

public static void main(String args[])

 

en la línea de código de arriba el arreglo de cadenas que contiene los argumentos en la línea de comandos es args.

 

class Promedio {

            public static void main(String args[]){

                        int sum=0;

                        for (int i = 0; i <  args.length; i++){

                                    sum+=Integer.parseInt(args[i]);

                        }

                        System.out.println(“La suma es :”+sum);

                        System.out.println(“El promedio es :”+

(float)sum / args.length);

            }

}

 

compilar con

 

javac Promedio.java

 

y ejecutar con

 

java Promedio 8 22 4 9 33

 

Los argumentos van después del nombre de la aplicación y son separados uno de otro por espacio. En la línea de arriba el nombre de la aplicación es Promedio y los argumentos son 8, 22, 4, 9 y 33.
